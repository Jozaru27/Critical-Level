Critical Level

Sobre Nosotros

¿En qué consiste?

Critical Level es una página web que busca como objetivo brindar tanto a jugadores casuales como a críticos un lugar donde poder expresarse y compartir su opinión sobre los títulos que hayan jugado, mientras comprueban las opiniones de otros usuarios.

¿Cómo funciona?

Además de nuestros propios ficheros, lógica de procesamiento, base de datos y librerías, utilizamos la API proporcionada por RAWG, obteniendo así todos los datos necesarios sobre los videojuegos para que la página funcione.
